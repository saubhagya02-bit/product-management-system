# Product Management System

A **full-stack web application** for managing products built using **Java Spring Boot**, **MySQL**, **React** and **Bootstrap**. This project is for learning purposes and demonstrates CRUD operations, REST API integration and frontend-backend communication.

## Features

- Add, edit, delete and view products
- Responsive UI with Bootstrap
- RESTful API with Spring Boot
- Persistent storage using MySQL

## Tech Stack

- **Backend:** Java, Spring Boot, Spring Data JPA, Hibernate, MySQL
- **Frontend:** React, Axios, Bootstrap
- **Database:** MySQL
